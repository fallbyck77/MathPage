1. ¿Qué es un Grupo?

Las bases del Álgebra Abstracta

Introducción

Nos vamos a adentrar a una de las ramas matemáticas, en la cual, como en la mayoría no requerimos fundamentalmente de los números para su desarrollo, aquí tratamos de cubrir conceptos más avanzados como las simetrías, las permutaciones, y naturalmente los patrones que podamos ir notando en la naturaleza. Definición Formal

Sea $G$ un conjunto no vacío y $*$ una operación binaria en $G$. El par $(G, *)$ es un grupo si satisface los siguientes axiomas:

  • (i) Asociatividad: Para todo $a, b, c \in G$, se cumple que $(a * b) * c = a * (b * c)$.
  • (ii) Elemento Identidad (o Neutro): Existe un elemento $e \in G$ tal que para todo $a \in G$, se cumple que $e * a = a * e = a$.
  • (iii) Elemento Inverso: Para cada $a \in G$, existe un elemento $a^{-1} \in G$ tal que $a * a^{-1} = a^{-1} * a = e$.

Notemos que la propiedad de Cierre (para todo $a, b \in G$, el resultado $a * b$ también está en $G$) está implícita en la definición de una operación binaria en $G$.

Si la operación $*$ es conmutativa, el grupo $(G, *)$ se denomina Abeliano. El orden de un grupo es el número de elementos en $G$, y se denota por $|G|$.

Cuando los matemáticos trabajan con un grupo genérico, la operación usualmente se suprime por conveniencia: se habla del "grupo $G$" y se escribe $ab$ o $a \cdot b$ para el resultado de combinar $a$ y $b$. El elemento $ab$ se denomina genéricamente el "producto" de $a$ y $b$.

Primeros Ejemplos

Aquí puedes detallar algunos ejemplos clásicos para dar una intuición de lo que es un grupo.

  • Los enteros con la suma: El conjunto de los números enteros, $(\mathbb{Z}, +)$, forma un grupo. El elemento identidad es $0$ y el inverso de $a$ es $-a$.
  • Los reales positivos con la multiplicación: El conjunto $(\mathbb{R}^+, \cdot)$ es un grupo. ¿Qué pasa con $(\mathbb{R}, \cdot)$? ¿Por qué el cero es un problema?
  • Un no-ejemplo: El conjunto de los números naturales $(\mathbb{N}, +)$. ¿Qué axioma no se cumple?

Propiedades Básicas

A partir de los axiomas, se pueden deducir varias propiedades importantes que todo grupo debe cumplir.

  • Unicidad del elemento identidad.
  • Unicidad del elemento inverso para cada elemento.
  • Las leyes de cancelación: si $a*b = a*c$, entonces $b=c$.
